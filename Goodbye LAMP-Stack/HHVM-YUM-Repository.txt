Zur Installation via YUM, dem Paket-Manager von Distributionen wie Fedora, muss das passende Repository konfiguriert werden:1.	Erzeugen der Datei /etc/yum.repos.d/hhvm.repo mit folgendem Inhalt:[hhvm] name=HHVM for Fedora $releasever - $basearch baseurl=http://dl.hhvm.com/fedora/$releasever/$basearch/enabled=1gpgcheck=1gpgkey=file:///etc/pki/rpm-gpg/hhvm.gpg.key2.	PG-Key herunterladen:wget -O/etc/pki/rpm-gpg/hhvm.gpg.key http://dl.hhvm.com/conf/hhvm.gpg.key
